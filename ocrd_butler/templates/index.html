Welcome Madam/Sir!

How can I help you today?

With this tool you can process tasks in the ecosystem of the OCR-D project.

You can explore the currently installed ORCD processors and take a look at the Chains that were built with these processors.

Currently there is no form to create chains and tasks directly in the UI.
But you can use the swagger interface of the API behind this site.
