Welcome Madam/Sir!

How can I help you today?

With this tool you can process tasks in the ecosystem of the OCR-D project.

You can explore the currently installed ORCD processors and take a look at the Chains that were built with these processors.

You can use the forms to create chains and tasks directly in the UI. But you also can use the swagger interface of the API behind this site.

On the other hand you can use the api to create chains and tasks programatically.

Chains


    curl -X POST -H "Content-Type: application/json" http://ocrd-butler.sbb.berlin/api/chains/chain -d '{
        "name": "Tesseract",
        "description": "Chain with only tesseract processors",
        "processors": [
            "ocrd-tesserocr-segment-region",
            "ocrd-tesserocr-segment-line",
            "ocrd-tesserocr-segment-word",
            "ocrd-tesserocr-recognize"
        ]
    }'

    

    curl -X POST -H "Content-Type: application/json" http://ocrd-butler.sbb.berlin/api/chains/chain -d '{
        "name": "Olena - Calamari",
        "description": "Chain with olena binarization and calamari recognize",
        "processors": [
            "ocrd-olena-binarize",
            "ocrd-tesserocr-segment-region",
            "ocrd-tesserocr-segment-line",
            "ocrd-calamari-recognize"
        ]
    }'

Tasks
